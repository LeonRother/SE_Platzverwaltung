# SWE Platzverwaltung
Repo für die Software Engineering-Vorlesung an der DHBW Karlsruhe. Gruppe 5:
Büro-Platzverwaltung als WebApp mit REST API Backend und Vue Frontend.

Techstack:
- Backend: C# ASP.NET 
- Frontend:
    - Vue.js
    - TailwindCSS
    - TypeScript (optional)
- Datenbank: sqlite
